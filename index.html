import { Scribe } from "@industrialscribe/sdk";

const scribe = new Scribe(process.env.SCRIBE_KEY);

const quote = await scribe.quote.create({
  part: "CNC bracket A-204",
  qty: 500,
  material: "AL6061",
  tolerance: "±0.05mm",

});

console.log(quote.confidence);    // "high"

Use cases

Every ERP module a SEA shop actually needs.

Drop-in primitives for the workflows that eat 80% of a manufacturing BD/operator's day.

QT

Quotation Engine

3-gate qualification, BOM-aware pricing, margin guardrails. Returns a quote row your sales team can send.
RC

RCA Generator

Feed it a defect report. It writes a 5-Why root cause analysis with corrective actions, audit-ready.
SO

SOP Writer

Voice-note in your process. Get back a complete, formatted SOP with control points and verification steps.
PR

Production Report

Pulls from your shop floor data. Generates the weekly production report your boss reads on Monday.

Briefs you for a high-stakes call. Counter-offers, BATNAs, and the exact words to say first.
SC

Scrap Tracker

Photo in, scrap reason out. Tracks rate by machine, operator, shift — all without a clipboard.
